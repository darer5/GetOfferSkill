# VibeGetOffer 工作流程指南

## 概述

VibeGetOffer 提供三种工作模式：完整流程、单独调用、断点续传。

## 模式 A：完整流程（4 阶段）

```
阶段1 → 阶段2 → 阶段3 → 阶段4
  JD分析 → 简历优化 → 面试练习 → 面试复盘
```

### 触发方式
- "开始求职流程"
- "一键开始"
- "完整流程"

### 执行步骤

**阶段 1：JD 分析**
- 调用技能：`jd-analysis`
- 输入：JD 文本或链接
- 输出：核心关键词、岗位痛点、面试预测、简历修改建议

**阶段 2：简历优化**
- 调用技能：`resume-coach`
- 输入：阶段 1 输出 + MyCareer 职业档案 + 用户提供的原始简历/参考简历（如有）
- 输出：可投递的 JD 定制简历 + 待确认清单

**阶段 3：面试练习**
- 调用技能：`interview-practice`
- 输入：JD + 优化后简历 + 公司/岗位信息
- 输出：多轮面试练习记录

**阶段 4：面试复盘**
- 调用技能：`interview-review`
- 输入：面试录音转写文本
- 输出：五维评分、风险分析、改进建议

---

## 模式 B：单独调用

直接使用指定技能，不经过完整流程。

| 技能 | 触发词 | 功能 |
|------|--------|------|
| jd-analysis | "分析 JD"、"看这个岗位" | JD 深度分析 |
| resume-coach | "优化简历"、"评估简历" | 简历优化 |
| interview-practice | "面试练习"、"模拟面试" | 面试演练 |
| interview-review | "复盘面试"、"分析面试" | 面试复盘 |

---

## 模式 C：断点续传

从指定节点开始，跳过前面的步骤。

| 起始节点 | 触发词 | 需要提供 |
|----------|--------|----------|
| 简历优化 | "从简历优化开始" | 简历内容或 MyCareer 档案；如需岗位定制则还需要 JD |
| 面试练习 | "从面试练习开始" | JD + 简历 |
| 面试复盘 | "从复盘开始" | 面试转写文本 |

---

## 技能详解

### jd-analysis

**功能**：深度分析岗位 JD，提取关键信息

**输出**：
- 核心关键词云
- 岗位核心使命
- 公司/团队痛点猜测
- 简历修改建议
- 面试必考题预测
- 项目经历关联表

### resume-coach

**功能**：简历评估与优化（ATS 友好）

**模式**：
- 无 JD：简历体检（五维评分）
- 有 JD：简历重塑（匹配度优化）

**输出**：通用优化建议，或可投递的 JD 定制简历

### interview-practice

**流程**：创建面试 → 多轮演练 → 即时反馈 → 生成报告

**轮次**：
1. 技术面（≥5.5/10 通过）
2. 业务面（≥6.0/10 通过）
3. 综合面（≥6.5/10 通过）
4. HR面（≥7.0/10 通过）

### interview-review

**流程**：文本解析 → Q&A 提取 → 深度诊断 → 报告生成

**输出**：
- 五维加权评分
- 风险系数建模
- 黄金改进话术
- ReviewQA 问题库
