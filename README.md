# GetOfferSkill

整合多个 AI 技能的求职辅助工具，帮助用户实现从 JD 分析 → 简历优化 → 面试训练 → 面试复盘的全流程支持。

## 核心能力

GetOfferSkill 整合以下 4 个技能：

| 技能 | 功能 | 触发词示例 |
|------|------|-----------|
| **jd-analysis** | 分析 JD，提取关键词、岗位痛点、面试预测 | "分析这个 JD"、"看这个岗位" |
| **resume-coach** | 评估和优化简历 | "优化简历"、"帮我看简历" |
| **interview-practice** | 面试模拟演练，多轮面试 + 即时反馈 | "开始面试练习"、"模拟面试" |
| **interview-review** | 面试复盘分析，五维评分 + 改进建议 | "复盘面试"、"面试评估" |

## 工作模式

### 模式 A：完整流程（4 阶段）

当用户说"开始求职流程"或"一键开始"时，按顺序执行：

```
[阶段1] jd-analysis 分析 JD
    ↓
[阶段2] resume-coach 优化简历
    ↓
[阶段3] interview-practice 面试练习
    ↓
[阶段4] interview-review 面试复盘
```

### 模式 B：单独调用

用户可直接调用指定技能。

### 模式 C：断点续传

用户可从指定节点继续。

## 快速开始

### 方式一：在 Claude Code 中使用

将 `CLAUDE.md` 的内容配置到你的 Claude Code 项目中，即可使用 GetOfferSkill 的所有技能。

### 方式二：单独使用某个技能

参考 `skills/` 目录下各技能的说明文档，复制对应的技能描述到你的 AI 助手中。

## 技能详情

### jd-analysis
分析岗位 JD，提取核心关键词、岗位痛点、核心诉求，并指导面试准备。

### resume-coach
AI 简历金牌教练 - 资深猎头&HR总监视角的简历优化专家。支持简历体检和简历重塑两大模块。

### interview-practice
具有持久记忆的智能面试陪练系统，支持多轮面试、STAR法则追问、即时反馈和面试记录持久化。

### interview-review
基于面试录音转写文本执行两阶段深度复盘诊断，涵盖职级动态识别、结构化Q&A提取、五维加权评分、风险系数建模及黄金改进话术生成。
